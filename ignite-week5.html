Problems

Pick one. Build it. Own it.

PS — 01

Smart Daily Planner

Build a daily planner where users can manage their tasks with priorities and filters.

  • Add tasks with a title
  • Set priority — High / Medium / Low
  • Mark tasks as completed
  • Delete tasks
  • Filter tasks by priority
⭐ Bonus Points
  • LocalStorage support
  • Smooth animations
  • Dark mode toggle
PS — 02

Student Expense Tracker

Create a web app to track income, expenses, and visualize spending patterns.

  • Add income entries
  • Add expense entries
  • Show total balance
  • Show expense breakdown
  • Bar graph using JS + CSS only
⭐ Bonus Points
  • Category-wise filtering
  • Animated UI
  • LocalStorage persistence
PS — 03

Interactive Quiz App

Build a quiz application with timer, scoring, and a detailed result page.

  • 5 multiple choice questions
  • Countdown timer
  • Score calculation
  • Result page with performance message
⭐ Bonus Points
  • Random question shuffle
  • Progress bar
  • Restart quiz feature
🌟 Special

Student Innovation Challenge

Got a unique idea? Build it your way. This track is for students who go beyond the standard problems.

  • Completely open-ended idea
  • HTML + CSS + JS only
  • Solve a real problem or be genuinely creative
  • Must be functional and demonstrable
